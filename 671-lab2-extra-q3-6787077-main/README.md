## Calculate the Slope of a Line
 
จงเขียนโปรแกรมเพื่อคำนวณหาความชันของเส้นตรงที่ลากเชื่อมจุด `P(x1,y1)` และจุด `Q(x2,y2)` โดยความชันของเส้นตรง แทนด้วย `m` สามารถหาจาก `m = y2 - y1 /x2 - x1`

โปรแกรมจะรับ input ค่า `x1`, `y1`, `x2`, และ `y2` ตามลำดับ จากนั้นโปรแกรมจะคำนวณความชันและแสดงผลลัพธ์ความชัน (ทศนิยม 2 ตำแหน่ง)

**Hint:** สามารถใช้คำสั่ง `round(number, 2)` ในการปรับ `number` ใด ๆ ให้เป็นจุดทศนิยม 2 ตำแหน่ง
ตัวอย่างการใช้การคำสั่ง: `round(12.666667,2)` จะได้ผลลัพธ์เป็น `12.67`
<hr>

**ตัวอย่างที่ 1:**

**Input:** `P(x1, y1)= (4,3)` และ `Q(x2, y2)= (2,1)`
```
4
3
2
1
```
**Expected output:** หลังจากคำนวณความชันแล้ว โปรแกรมควรแสดงผลลัพธ์เป็น 
```
This slope is 1.0
```
<hr>

**ตัวอย่างที่ 2:**

**Input:** `P(x1, y1)= (8,10)` และ `Q(x2, y2)= (-8,-10)`
```
8 
10
-8 
-10 
```
**Expected output:** หลังจากคำนวณความชันแล้ว โปรแกรมควรแสดงผลลัพธ์เป็น 
```
This slope is 1.25
```
<hr>

7
