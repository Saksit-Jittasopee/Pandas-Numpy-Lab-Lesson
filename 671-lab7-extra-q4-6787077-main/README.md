## List Insert and Update

จงเขียนโปรแกรมที่รับตัวเลข `5` จำนวน โดยใช้ **loop** โดยตัวเลขที่รับเข้ามาจะถูกเก็บไว้ใน list `A` จากนั้นรับ input `x` จากผู้ใช้ เปรียบเทียบตัวเลขแต่ละตัวใน list กับ `x` และแทนตัวเลขใน list `A` ด้วยตัวอักษรตาม condition ดังต่อไปนี้

* ถ้าตัวเลขมากกว่า `x` ให้แทนตัวเลขนั้นด้วยตัวอักษร `g`
* ถ้าตัวเลขน้อยกว่า `x` ให้แทนตัวเลขนั้นด้วยตัวอักษร `s`
* ถ้าตัวเลขเท่ากับ `x` ให้แทนตัวเลขนั้นด้วยตัวอักษร `e`

จากนั้น print list ที่ได้รับการ update แล้ว

**หมายเหตุ:** 
นักศึกษาสามารถสมมติได้ว่าผู้ใช้จะ input ตัวเลขเท่านั้น

<hr>

**ตัวอย่างที่ 1:**
**Input:** `A = [1, 21, 14, 5.5, -3]` และ `x = 5.5` 
```
1
21
14
5.5
-3
5.5
```
**Expected output:** โปรแกรมจะแสดงค่าใน list ดังต่อไปนี้
```
['s', 'g', 'g', 'e', 's']
```
<hr>
