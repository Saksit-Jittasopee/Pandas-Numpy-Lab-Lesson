## Series of Numbers
จงเขียนโปรแกรมเพื่อแสดงซีรี่ย์ของตัวเลขดังต่อไปนี้ `x,xx,xxx,xxxx,...` ทั้งหมด `y` term นักศึกษาจะต้องใช้ **for loop** ในการทำโจทย์ข้อนี้

โปรแกรมจะรับ 2 input: `x` คือตัวเลขที่จะแสดง และ`y` ซึ่งคือจำนวน term


**หมายเหตุ:** 
* นักศึกษาสามารถสมมติได้ว่าผู้ใช้จะ input เฉพาะตัวเลขจำนวนเต็มที่ มากกว่า 0 และ `x` เป็นเลข 1-9 เท่านั้น


<hr>

**ตัวอย่างที่ 1:**
**Input:** `x=3` และ `y=5`
```
3
5
```
**Expected output:** โปรแกรมจะแสดงผลลัพธ์ดังต่อไปนี้ โดยระหว่างแต่ละ term ของตัวเลขจะขั้นโดย `,` และไม่มี space(s) ระหว่าง term
```
3,33,333,3333,33333
```
<hr>

**ตัวอย่างที่ 2:**
**Input:** `x=1` และ `y=8`
```
1
8
```
**Expected output:** โปรแกรมจะแสดงผลลัพธ์ดังต่อไปนี้ โดยระหว่างแต่ละ term ของตัวเลขจะขั้นโดย `,` และไม่มี space(s) ระหว่าง term
```
1,11,111,1111,11111,111111,1111111,11111111
```
<hr>

**ตัวอย่างที่ 3:**
**Input:** `x=9` และ `y=1`
```
9
1
```
**Expected output:** โปรแกรมจะแสดงผลลัพธ์ดังต่อไปนี้ โดยระหว่างแต่ละ term ของตัวเลขจะขั้นโดย `,` และไม่มี space(s) ระหว่าง term
```
9
```
