## Matrix Multiplication

จงเขียนโปรแกรมเพื่อรับ input ตัวเลขจำนวนเต็ม  `n` จากนั้นรับตัวเลขเพื่อสร้าง matrix A และ matrix B ขนาด nxn คำนวณหาผลคูณของ matrix A และ B

วิธีการคูณ matrix สามารถดูได้จาก link นี้ https://matrix.reshish.com/multiplication.php

**หมายเหตุ:** นักศึกษาสามารถสมมติได้ว่าผู้ใช้จะ input ตัวเลขจำนวนเต็มเท่านั้น และ `n` มากกว่า 0
<hr>

**ตัวอย่างที่ 1:**

**Input:** ผู้ใช้ input `n=3`, `A=[[1, 2, 3], [4, 5, 6], [7, 8, 9]]` และ`B=[[1, 2, 3], [1, 2, 3], [1, 2, 3]]`
```
3
1
2
3
4
5
6
7
8
9
1
2
3
1
2
3
1
2
3
```
**Expected output:** โปรแกรมจะแสดงผลลัพธ์ดังนี้ ประกอบไปด้วย 2D list แสดง matrix `A`,`B` และผลคูณของทั้งสอง matrix
```
[[1, 2, 3], [4, 5, 6], [7, 8, 9]]
[[1, 2, 3], [1, 2, 3], [1, 2, 3]]
[[6, 12, 18], [15, 30, 45], [24, 48, 72]]
```
<hr>

**ตัวอย่างที่ 2:**

**Input:** ผู้ใช้ input `n=2`, `A=[[1, 4], [-3, 2]]` และ`[[5, 3], [2, 1]]`
```
2
1
4
-3
2
5
3
2
1
```
**Expected output:** โปรแกรมจะแสดงผลลัพธ์ดังนี้ ประกอบไปด้วย 2D list แสดง matrix `A`,`B` และผลคูณของทั้งสอง matrix
```
[[1, 4], [-3, 2]]
[[5, 3], [2, 1]]
[[13, 7], [-11, 7]]
```
<hr>

**ตัวอย่างที่ 3:**

**Input:** ผู้ใช้ input `n=1`, `A=[[5]]` และ`[[3]]`
```
1
5
3
```
**Expected output:** โปรแกรมจะแสดงผลลัพธ์ดังนี้ ประกอบไปด้วย 2D list แสดง matrix `A`,`B` และผลคูณของทั้งสอง matrix
```
[[5]]
[[3]]
[[15]]
```
