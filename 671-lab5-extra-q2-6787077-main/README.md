## Find Greatest Common Divisor

จงเขียนโปรแกรมเพื่อหา"หารร่วมมาก" ของจำนวนเต็ม 2 ตัวที่ไม่ใช่ 0: `a`และ`b` โดย `a<b`  หารร่วมมาก หรือ ห.ร.ม.คือ จำนวนเต็มที่มากที่สุด ที่สามารถนําไปหารจำนวน `a`และ`b`ได้ลงตัว

โปรแกรมรับ input เป็นจำนวนเต็ม 2 จำนวน จากนั้นคำนวณห.ร.ม.ของเลข 2 จำนวนนี้และแสดงผลลัพธ์ หากจำนวนทั้งสองไม่มีตัวเลขใด ๆ ที่สามารถหารร่วมกันได้ (เช่น 2 และ 3) ห.ร.ม จะมีค่าเป็น 1

**หมายเหตุ:** 
* นักศึกษาสามารถสมมติได้ว่าผู้ใช้จะ input เป็นตัวเลขจำนวนเต็ม `a`และ`b`โดยทั้งคู่มีค่ามากกว่า 0 และ `a<b` 
* สามารถเลือกได้ว่าจะใช้ **for** หรือ **while** loop ในการแก้ปัญหานี้
* เนื่องจากเป็นการฝึกกระบวนการคิด ไม่อนุญาตให้ใช้ built-in functions ใด ๆ ที่ทำให้เกิดผลลัพธ์ทันที ในการแก้ปัญหานี้
<hr>

**ตัวอย่างที่ 1:**

**Input:** `a=3`และ`b=3`
```
3
3
```
**Expected output:** หลังจากคำนวณแล้ว โปรแกรมจะแสดงผลลัพธ์ดังนี้ โดยห.ร.มของทั้ง 2 ตัวเลขคือ
```
3
```
<hr>

**ตัวอย่างที่ 2:**

```

3
```

**Expected output:** หลังจากคำนวณแล้ว โปรแกรมจะแสดงผลลัพธ์ดังนี้ โดยห.ร.มของทั้ง 2 ตัวเลขคือ
```
1
```

```
24
36
```
