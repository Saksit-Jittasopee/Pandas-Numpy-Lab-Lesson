## Calculate 3D Distance 

จงเขียนโปรแกรมที่คำนวณระยะทาง 3D distance โดยการเรียกใช้ function ที่ชื่อว่า `cal_dist` เพื่อการคำนวณ

โดยกำหนดให้ function `cal_dist` นี้ จะรับค่า parameter 6 ค่านั้นคือ `x1`, `y1`, `z1`, `x2`, `y2`, และ `z2` ตามลำดับ และมีค่า default parameters ดังนี้ `x1=7.0`, `y1=4.0`, `z1=3.0`, `x2=17.0`, `y2=6.0`, และ `z2=2.0`

การทำงานของโปรแกรมจะเริ่มจากการรับค่า parameter `x1`, `y1`, `z1`, `x2`, `y2`, และ `z2` จากผู้ใช้ โดยผู้ใช้ input ได้เป็นตัวเลข หรือ `q` เท่านั้น 
ภายใน `cal_dist` function จะมีการคำนวณระยะทางจากสมการดังนี้
```
dist = sqrt((x1-x2)^2 + (y1-y2)^2 + (z1-z2)^2)
```
* ถ้าผู้ใช้ใส่ค่า `q` ใน parameter `x1`, `y1`, หรือ `z1` ตัวใดตัวนึง โปรแกรมจะใช้ค่า default parameters ของ `x1`, `y1`, `z1` ทั้งหมดในการคำนวณ (`x1=7.0`, `y1=4.0`, และ `z1=3.0`)
* ถ้าผู้ใช้ใส่ค่า `q` ใน parameter `x2`, `y2`, หรือ `z2` ให้ใช้ค่า โปรแกรมจะใช้ค่า default parameters ของ `x1`, `y1`, `z1` ทั้งหมดในการคำนวณ (`x2=17.0`, `y2=6.0`, และ `z2=2.0`)
* ถ้าผู้ใช้ใส่ค่า `q` ใน parameter `x1`, `y1`, หรือ `z1` และ `x2`, `y2`, หรือ `z2` ให้ใช้ค่า โปรแกรมจะใช้ค่า default parameters ทั้งหมดในการคำนวณ (`x1=7.0`, `y1=4.0`, `z1=3.0`, `x2=17.0`, `y2=6.0`, และ `z2=2.0`)

หลังจากได้ค่าระยะทางแล้ว `cal_dist` function จะทำการ return ค่ากลับไปเพื่อแสดงผลลัพธ์เป็นเลขทศนิยม 2 หลัก 

**Note:** 
* สามารถใช้ math.pow(x,2) ในการคำนวณเลขยกกำลังได้
* สามารถใช้ math.sqrt(x) ในการคำนวณ square root ได้
* หากไม่ได้ใช้ `cal_dist` function เพื่อการคำนวณ จะไม่ได้คะแนนจากข้อนี้ 

**ตัวอย่างที่ 1:**
**Input:** `x1=1, y1=2, z1=q, x2=1, y2=2, z2=q`  
```
1
2
q
1
2
q
```
**Expected output:** โปรแกรมจะแสดงค่า ระยะทางดังต่อไปนี้
```
10.25
```
**คำอธิบายและวิธีการคำนวณ:** <br>
เนื่องจาก z1 และ z2 เป็น q <br>
ดังนั้นจุด p1 (`x1`, `y1`, `z1`) และ p2 (`x2`, `y2`, `z2`) จะใช้ค่า default parameters <br>
(`x1=7.0`, `y1=4.0`, และ `z1=3.0`)<br>
(`x2=17.0`, `y2=6.0`, และ `z2=2.0`)<br>
ดังนั้น <br>
```
dist = sqrt((7-17)^2 + (4-6)^2 + (3-2)^2)
    =  sqrt(100 + 4 + 1)
    =  sqrt(105)
    =  10.25
```

**ตัวอย่างที่ 2:**
**Input:** `x1=1, y1=2, z1=3, x2=q, y2=1, z2=2`  
```
1
2
3
q
1
2
```
**Expected output:** โปรแกรมจะแสดงค่า ระยะทางดังต่อไปนี้
```
16.52
```
**คำอธิบายและวิธีการคำนวณ:** <br>
เนื่องจาก x2 เป็น q <br>
ดังนั้นจุด  p2 (`x2`, `y2`, `z2`) จะใช้ค่า default parameters <br>
(`x1=1`, `y1=2`, และ `z1=3`)<br>
(`x2=17.0`, `y2=6.0`, และ `z2=2.0`)<br>
ดังนั้น <br>
```
dist = sqrt((1-17)^2 + (2-6)^2 + (3-2)^2)
    =  sqrt(256 + 16 + 1)
    =  sqrt(273)
    =  16.52
```
